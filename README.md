# CAP Observer Log

A mobile-friendly mission observer logging application for Civil Air Patrol operations.

## Features

- **Mission Info** - Track sortie numbers, aircraft, crew assignments
- **Times** - Log all mission times (brief, takeoff, landing, debrief)
- **Check-ins** - Record position reports with timestamps
- **Events** - Log observations with GPS coordinates and CAP grid
- **CO Comms** - Colorado Wing TDFM channels and callsign roster
- **Emergency** - C182T emergency procedures quick reference
- **Reference** - Radio frequencies, reporting requirements, G1000 reference

## Usage

Access the app at: `https://YOUR_USERNAME.github.io/cap-observer-log/`

Works offline once loaded (add to home screen for best experience).

## Deployment

1. Fork this repository
2. Enable GitHub Pages in Settings → Pages → Source: main branch
3. Access at your GitHub Pages URL
